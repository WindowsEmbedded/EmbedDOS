# EmbeddedDOS

#### 介绍
EmbeddedDOS是一个作者闲着没事在2022年2月份创作的一个伪系统
欢迎参与贡献



#### 使用说明

1.去“发行版”页面随意下载一个版本
2.点击“sys.exe”（2.0版本为“maint.exe”）

#### 参与贡献

1.  Fork 本仓库
2.  新建 Feat_xxx 分支
3.  提交代码
4.  新建 Pull Request
